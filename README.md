## QR Code Generator

A simple QR Code Generator built using HTML, CSS, and JavaScript. It converts text or URLs into a QR code instantly using an external QR API.

## Features

Generate QR codes from text or URLs
Instant QR code preview
Input validation with error feedback
Clean and minimal UI

## Built With

HTML
CSS
JavaScript
QR Code API

## What I Learned

Working with external APIs
Handling user input validation
Dynamically updating images in the DOM
Managing UI states with JavaScript

## How to Use

Enter text or a URL
Click Generate QR Code
Scan or download the generated QR code
